═══════════════════════════════════════════════════════════════════
  ⚠️  PROBLÈMES COURANTS
═══════════════════════════════════════════════════════════════════

❌ Dashboard ne charge pas
   → Attends 60 secondes (Render se réveille)
   → Vérifie que l'API répond : curl https://scotty-dashboard-api.onrender.com/api/weather

❌ Modifications pas visibles
   → Vide le cache : Cmd + Shift + R
   → Teste en navigation privée : Cmd + Shift + N
   → Vérifie que le git push a fonctionné

❌ Statistiques à 0
   → Problème de cache JS
   → Change la version dans dashboard.html (?v=20251016)
   → Vide le cache du navigateur

❌ Météo ne s'affiche pas
   → Vérifie la variable OPENWEATHER_API_KEY sur Render
   → Teste : curl https://scotty-dashboard-api.onrender.com/api/weather
